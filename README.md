# 🎨 CSS Sandbox

A growing collection of HTML, CSS & JavaScript mini-projects and design experiments — built to accompany the tutorials I post on TikTok. Every new project I create gets added here so my followers can grab the code, learn from it, and remix it.

## 📌 About

This repo is my public sandbox. Each folder is a standalone project — a UI design, animation, layout experiment, or small tool — usually inspired by or featured in one of my TikTok videos. No two projects are related; think of this as a running archive rather than a single app.

## 🛠️ Tech Stack

- **HTML5** – structure
- **CSS3** – styling, layout, animations
- **JavaScript (Vanilla)** – interactivity, where needed

## 🚀 How to Use

1. Clone the repo:
   ```bash
   git clone https://github.com/MuhammadHaroon5/my-css-sandbox
   ```
2. Open the project folder you're interested in.
3. Open `index.html` in your browser (or use a live server extension) — that's it, no build step needed.
